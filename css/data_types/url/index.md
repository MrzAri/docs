{{Summary_Section|The <code>&lt;url></code> CSS data type, formerly called the <code>&lt;uri></code> data type, represents a reference to a file or fragment.  It is specified with the <code>url()</code> function.}}
{{Data_Type_Page
|Content=The <code>url(string)</code> function identifies the parameter string as a resource identifier; the string may be quoted but need not be (so long as it doesn't contain any internal whitespace).

Not all references in CSS to external files require the use of the <code>url()</code> syntax; file names in [[css/atrules/@import |<code>@import</code> rules]], for example, may be given as simple quoted [[css/data_types/string| <code>&lt;string></code>]] values.
