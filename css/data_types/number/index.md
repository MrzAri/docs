{{Summary_Section|The <code>&lt;number></code> CSS data type represents positive or negative decimal or integer values.}}
{{Data_Type_Page
|Content=Numbers in the [http://www.w3.org/TR/css3-values/#numbers CSS definition] can only accept ordinary decimal numbers: a series of digits optionally followed by a decimal point ('''.''') and more digits.   This differs from the [http://www.w3.org/TR/SVG11/types.html#DataTypeNumber SVG definition] which allows for numbers in attribute properties to be specified using scientific notation (such as '''1.3e-5''' for '''0.000013''').  If an SVG presentational attribute is specified using CSS styles, it must conform to the CSS data type.
