===Remarks===
The ''':root''' pseudo-class is a structural pseudo-class. Structural pseudo-classes enable selection based on extra information in the document tree that can't be selected using simple selectors or combinators.
|Import_Notes=
===Syntax===

selector
:root
===Parameters===
;''selector'':A CSS simple selector.
