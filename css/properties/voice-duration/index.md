{{Summary_Section|The <code>voice-duration</code> property allows the author to explicitly set the amount of time it should take a speech synthesizer to read an element's content, for example to allow the speech to be synchronized with other media.  With a value of  <code>auto</code> (the default) the length of time it takes to read the content is determined by the content itself and the [[css/properties/voice-rate|<code>voice-rate</code>]] property.}}
