# line-break

## Summary

Deprecated. Gets or sets line-breaking rules for text in selected languages such as Japanese, Chinese, and Korean.

## Overview table

[Initial value](/css/concepts/initial_value)
:   `auto`
Applies to
:   All elements
[Inherited](/css/concepts/inherited)
:   Yes
Media
:   visual
[Computed value](/css/concepts/computed_value)
:   specified value
Animatable
:   No
[CSS Object Model Property](/css/concepts/cssom)
:   ``

## Syntax

-   `line-break: normal`
-   `line-break: strict`

## Values

normal
:   Default. Normal line-breaking mode is applied.

strict
:   Stricter line-breaking rules are enforced.

## Notes

### Remarks

This property is deprecated. It is replaced by the [**-ms-word-break**](/css/properties/word-break) property starting in Windows Internet ExplorerÂ 7.
