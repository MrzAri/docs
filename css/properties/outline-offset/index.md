== Summary ==
Sets the offset of the outline frame outside of the border edge.

== Syntax ==

<code>outline-offset: [number and unit]</code>

== Values ==

Floating-point number, followed by an absolute units designator (<code>cm</code>, <code>mm</code>, <code>in</code>, <code>pt</code>, or <code>pc</code>) or a relative units designator (<code>em</code>, <code>ex</code>, or <code>px</code>). For more information about the supported length units, see CSS Values and [[css/units/length|Units Reference (Length)]].

The property expects a floating point number followed by a length units designator ().

Default value: 0
