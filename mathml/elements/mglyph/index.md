{{Summary_Section|The MathML '''mglyph''' element is used to display non-standard symbols where existing Unicode characters are not available. It may be used within token elements.}}
{{Examples_Section
|Not_required=No
|Examples={{Single Example
|Language=HTML
|Description=This example demonstrates a simple usage of the mglyph element:
|Code=<syntaxhighlight lang="html5">
<math>
  <mi><mglyph src="my-glyph.png" alt="my glyph"/></mi>
</math>
</syntaxhighlight>
}}
