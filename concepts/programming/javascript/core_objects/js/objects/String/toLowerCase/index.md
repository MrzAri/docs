=== Summary ===
Returns value of a string converted to lowercase while keeping the original string intact.

=== Syntax ===
<syntaxhighlight lang="javascript">
"string".toLowerCase
</syntaxhighlight>

=== Parameters ===
None

=== Example ===
<syntaxhighlight lang="javascript">
var userInput="MixEDString";
document.write(userInput.toLowerCase());
</syntaxhighlight>
