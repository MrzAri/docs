# reload

## Summary

Reloads the current document.

*Method of [dom/Location](/dom/Location)*

## Syntax

``` {.js}
 location.reload(ignoreCache);
```

## Parameters

### ignoreCache

 Data-typeÂ

 Whether to ignore the cache and reload the page from scratch.
