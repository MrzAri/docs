# hostname

## Summary

Sets or retrieves the host name part of the location or URL.

<span data-meta="applies_to" data-type="key">Property of <span data-type="value">[dom/Location](/dom/Location)</span></span>

## Syntax

``` {.js}
var hostName = location.hostname;
location.hostname = hostName;
```

## Return Value

<span data-meta="return" data-type="key">Returns an object of type <span data-type="value">String</span></span>

The host name component of the URL.

**Needs Examples**: This section should include examples.

## Notes

If no host name is available, this property returns an empty string.
