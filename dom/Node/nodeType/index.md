{{API_Name}}
{{Summary_Section|Retrieves the type of the requested node.}}
{{API_Object_Property
|Property_applies_to=dom/Node
|Read_only=Yes
|Example_object_name=node
|Return_value_name=nodeType
|Javascript_data_type=Number
|Return_value_description=One of the node type constants defined on [[dom/Node|Node]], or null.
}}
{{Examples_Section
|Not_required=No
|Examples={{Single Example
|Language=JavaScript
|Description=This example assigns the '''nodeType''' property of the '''body''' object to a variable.
|Code=var iType {{=}} document.body.nodeType;
}}{{Single Example
|Language=JavaScript
|Description=This example assigns the '''nodeType''' property of a node created with the [[dom/methods/createElement|'''createElement''']] method to a variable.
|Code=var oNode {{=}} document.createElement("B");
document.body.insertBefore(oNode);
var iType {{=}} oNode.nodeType;
}}
}}
{{Notes_Section
|Notes=If the node represents an attribute retrieved from the [[dom/properties/attribute|'''attributes''']] collection, the '''nodeType''' returns <code>null</code>.
}}
