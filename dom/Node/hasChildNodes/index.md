===Remarks===
If the object contains HTML Elements or [[dom/TextNode|'''TextNodes''']], they can be accessed from the 
[[dom/properties/childNodes|'''childNodes''']] collection.
In Microsoft Internet Explorer 6, This method now applies to the [[dom/attributes|'''attribute''']] object.
