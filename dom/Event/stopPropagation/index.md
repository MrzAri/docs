{{Notes_Section
|Usage=Use this method to stop event propagation in the capturing or bubbling event phase.
|Notes=The event dispatches to all event listeners on the current target (regardless of capturing or bubbling) before the event flow stops. To completely prevent any remaining handlers from running, use the [[dom/methods/stopImmediatePropagation|'''stopImmediatePropagation''']] method instead.
