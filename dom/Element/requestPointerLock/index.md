# requestPointerLock

## Summary

requestPointerLock lets you lock the target of mouse events to a single element while hiding the cursor.

*Method of [dom/Element](/dom/Element)*

## Syntax

``` {.js}
 element.requestPointerLock();
```

## Return Value

No return value
