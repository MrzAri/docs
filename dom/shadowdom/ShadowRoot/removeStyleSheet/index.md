# removeStyleSheet

## Summary

Removes a style sheet from the ShadowRoot.

*Method of [dom/shadowdom/ShadowRoot](/dom/shadowdom/ShadowRoot)*

## Syntax

``` {.js}
var result = element.removeStyleSheet();
```

## Return Value

Returns an object of type StyleSheet.
