## Notes

### Remarks

The **location** property is not set during [**onkeypress**](/dom/KeyboardEvent/keypress) events. It is available only during [**onkeydown**](/dom/KeyboardEvent/keydown) and [**onkeyup**](/dom/KeyboardEvent/keyup) events.
