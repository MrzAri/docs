|Notes====Remarks===
The '''charCode''' property represents the character code during the [[dom/KeyboardEvent/keypress|'''onkeypress''']] event and the unmodified scan code of the key during [[dom/KeyboardEvent/keydown|'''onkeydown''']] and [[dom/KeyboardEvent/keyup|'''onkeyup''']] events. To detect whether the Alt, Ctrl, Meta, or Shift key is also pressed, see the [[dom/KeyboardEvent/altKey|'''altKey''']], [[dom/KeyboardEvent/ctrlKey|'''ctrlKey''']], [[dom/KeyboardEvent/metaKey|'''metaKey''']], or [[dom/KeyboardEvent/shiftKey|'''shiftKey''']] property of the event.
The '''charCode''' property  is provided for compatibility only. The latest version of the Document Object Model (DOM) Events specification defines a [[dom/KeyboardEvent/key|'''key''']] property instead.
