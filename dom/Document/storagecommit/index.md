===Remarks===
Local storage is saved to disk asynchronously to improve performance. The '''onstoragecommit''' event can be used to determine when the disk storage has been updated.
To invoke this event, do one of the following:
*Set or update a local storage value with [[apis/web-storage/methods/setItem|'''setItem''']].
*Remove a local storage value with [[apis/web-storage/methods/removeItem|'''removeItem''']].
