## Notes

### Remarks

The **onrowexit** event only fires on databound objects. This event applies to objects that identify themselves as data providers. Signals that the row in the databound object is about to be changed. To invoke this event, do one of the following:

-   Change rows in the data source.

The *pEvtObj* parameter is required for the following interfaces:

-   **HTMLAnchorEvents2**
-   **HTMLAreaEvents2**
-   **HTMLButtonElementEvents2**
-   **HTMLControlElementEvents2**
-   **HTMLDocumentEvents2**
-   **HTMLElementEvents2**
