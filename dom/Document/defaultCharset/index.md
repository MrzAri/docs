===Remarks===
The value depends on the current regional language settings. For typical settings in North America, the value is <code>windows-1252</code>.  For a list of other possible language settings, see [ie.Character_Set_Recognition#unknown_681#unknown_681 Charsets in Microsoft Internet Explorer 5].
The '''defaultCharset''' method is not implemented; therefore, the property is read-only.
