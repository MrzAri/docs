===Remarks===
When the [[html/attributes/value (HTMLProgressElement)|'''value''']] attribute is omitted from a [[html/elements/progress|'''progress''']] element, the element is indeterminate. The '''progress''' element shows activity, but not how much progress has actually been made.
If the [[html/attributes/value (HTMLProgressElement)|'''value''']] attribute is used without a [[html/attributes/max(HTMLProgressElement)|'''max''']] value, the range is from 0 to 1.
