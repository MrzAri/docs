|Notes====Remarks===
The '''tBody''', '''tHead''', and '''tFoot''' sections are mutually exclusive, so a '''tr''' is always contained in one of these sections and in the [[html/elements/table|'''table''']]. The [[dom/HTMLElement/rowIndex|'''rowIndex''']] property indicates the position of the object in the [[dom/HTMLElement/rows|'''rows''']] collection for the '''table'''. Collection indexes are in source order of the HTML document.
