===Remarks===
The '''tBody''', '''tHead''', and '''tFoot''' sections are mutually exclusive, so a '''tr''' is always contained in one of these sections and in the [[html/elements/table|'''table''']]. The [[dom/properties/rowIndex|'''rowIndex''']] property indicates the position of the object in the [[dom/properties/rows (table)|'''rows''']] collection for the '''table'''. Collection indexes are in source order of the HTML document.
