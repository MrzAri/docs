# HTMLMediaError

## Summary

Represents a media element (audio or video) error.

## Properties

API Name
:   Summary
[code](/dom/HTMLMediaError/code)
:   Returns the current **HTMLMediaError** code or null if no error has occurred.

## Methods

*No methods.*
