===Remarks===
When a '''length''' is provided, its value is added to the inter-character spacing value specified by the [[css/properties/letter-spacing|'''letterSpacing''']] property.
If a '''length''' is provided without a unit identifier (for example, an unqualified number such as 128), Internet Explorer processes the '''length''' as a width value.
