===Remarks===

The outermost [[svg/objects/SVGElement|'''svg''']] element is the furthest ancestor '''svg''' element in the current document fragment.
When the '''currentScale''' property  is called on an element other than the outermost [[svg/objects/SVGElement|'''svg''']] element, this property's behavior is undefined.
