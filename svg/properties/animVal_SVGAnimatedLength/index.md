===Remarks===

Internet Explorer 9 supports script-based SVG animation but  it does not support declarative-based SVG animation.  As a result,  the '''animVal''' property contains the same value as the  [[svg/properties/baseVal (SVGAnimatedLength)|'''baseVal''']] property.
