# baseVal (SVGAnimatedNumberList)

## Notes

### Remarks

Windows Internet Explorer 9 supports script-based SVG animation but it does not support declarative-based SVG animation. As a result, the [**animVal**](/svg/properties/animVal_(SVGAnimatedNumberList)) property contains the same value as the **baseVal** property.
