|Notes====Remarks===
If the '''align''' property is not set on the object, the table is aligned to the left edge of the available space as a block element.  If the property is set to '''left''' or '''right''', however, the table is aligned as a "floating" object.  Successive text and other elements flow to the right of the table if '''align''' is set to '''left''', or to the left of the table if '''align''' is set to '''right'''.  For more information, see the [[css/properties/float|'''styleFloat''']] property.
