When read as a Document Object Model (DOM) attribute,  '''background''' returns an absolute URL.  The value specified by the page author is returned when '''background''' is read as a content attribute, when the page is displayed in an earlier document compatibility mode, or when the page is viewed with an earlier version of the browser.
