|Notes====Remarks===
The '''legend''' tag is used only within a '''fieldSet''' tag. The contents of the '''legend''' tag are displayed by overwriting the information in the top border of the '''fieldSet'''.
The '''caption''' tag is used only within a [[html/elements/table|'''table''']] tag. Its contents are displayed centered above the table and do not interact with the table border.
