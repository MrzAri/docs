{{Summary_Section|The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format).}}
{{Markup_Attribute
|Applies_to=[[dom/HTMLInputElement|HTMLInputElement]]
|Property_applies_to=dom/HTMLElement
|Content=The short hint is displayed in the input field before the user enters a value.
}}
{{Examples_Section
|Not_required=No
|Examples={{Single Example
|Language=HTML
|Code=Address: <input type="text" name="address" placeholder="Insert here your address" />
Date of birth: <input type="text" name="day" placeholder="DD" />
<input type="text" name="month" placeholder="MM" />
<input type="text" name="year" placeholder="YYYY" />
}}
}}
{{Notes_Section
|Notes====Remarks===
The '''placeholder''' attribute can be used on text or text area input controls, or with text-based input attributes '''URL''' or '''email'''. The following example shows the placeholder text on a URL input field.
