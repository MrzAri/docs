|Notes====Remarks===
If the document is not in atomic selection mode, the '''ATOMICSELECTION''' attribute is ignored. To turn on the atomic selection mode, run the IDM_ATOMICSELECTION command with the value of <code>true</code>.
A document that is in atomic selection mode has slower performance when a selection is made anywhere in the document.
