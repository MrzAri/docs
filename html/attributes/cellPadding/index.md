# cellPadding

## Summary

Specifies the space, in pixels, between the cell wall and the cell content. Not supported in HTML5. Use CSS.

<table>
cellpadding attribute is not supported in HTML5. Use CSS instead. The cellpadding attribute specifies the space, in pixels, between the cell wall and the cell content. Note: Do not confuse this with the cellspacing attribute, which specifies the space between cells. Tip: It may be better to NOT specify a cellpadding, and use CSS to apply padding instead.
