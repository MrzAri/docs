{{Summary_Section|The '''dir''' element <code>&lt;dir></code> is used to list directory titles.

The '''dir''' element is deprecated in HTML 4.01, and obsolete in HTML5. Use CSS and the [[html/elements/ul|'''ul''' element]] instead.
}}
