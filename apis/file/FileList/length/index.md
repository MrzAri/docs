===Remarks===
Returns the number of files that are selected and available on the [[apis/file/FileList|'''FileList''']] object. This value can be greater than or equal to 1 if multiple file selection is enabled (typically via the <code>multiple</code> attribute on the <code>input</code> element). If no files have been selected, 0 is returned.
