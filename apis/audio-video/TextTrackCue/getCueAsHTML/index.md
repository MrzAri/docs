===Remarks===
The following example gets the list of [[apis/audio-video/properties/cues|'''cues''']] from a '''track''' element, and uses the [[apis/audio-video/properties/activeCues|'''activeCues''']] property to get the current cue. The '''getCueAsHTML''' method is used to display the caption in a '''div''' element below the video.
'''Note'''  To create timed text files in both Web Video Text Track (WebVTT) and Timed Text Markup Language (TTML) formats, see [http://go.microsoft.com/fwlink/p/?LinkID{{=}}251121 HTML5 Video Caption Maker] on the Windows Internet Explorer test drive site.
