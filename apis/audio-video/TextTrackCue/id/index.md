===Remarks===
This property is not implemented for the [[apis/audio-video/AudioTrack|'''AudioTrack''']] object.
The following example displays [[apis/audio-video/properties/startTime|'''startTime''']]and [[apis/audio-video/properties/endTime|'''endTime''']], [[apis/audio-video/properties/pauseOnExit|'''pauseOnExit''']],'''id''', and cue text for all cues in a track.
