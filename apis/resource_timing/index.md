# resource timing

## Summary

An interface for web applications to access the complete timing information for resources in a document.

API Name
:   Summary
[Performance](/apis/resource_timing/Performance)
:   Provides methods for performance timing.
[PerformanceResourceTiming](/apis/resource_timing/PerformanceResourceTiming)
:   Includes all resources fetched from the networking layer by the current browsing context.
