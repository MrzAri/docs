|'''HTMLStorage'''
|Represents the list of key/value pairs that have been assigned to a single storage area.
|-
|[[apis/web-storage/methods/clear|'''clear''']]
|Removes all key/value pairs from the Web Storage area.
|-
|[[apis/web-storage/methods/getItem|'''getItem''']]
|Retrieves the current value associated with the Web Storage key.
|-
|[[apis/web-storage/methods/initStorageEvent|'''initStorageEvent''']]
|Initializes a new Document Object Model (DOM) storage event  that the [[canvas/methods/createEvent|'''createEvent''']] method created.
|-
|[[apis/web-storage/methods/key|'''key''']]
|Retrieves the key at the specified index in the collection.
|-
|[[apis/web-storage/methods/removeItem|'''removeItem''']]
|Deletes a key/value pair from the Web Storage collection.
|-
|[[apis/web-storage/methods/setItem|'''setItem''']]
|Sets a key/value pair.
|-
|[[apis/web-storage/properties/key|'''key''']]
|Gets  the key that  is  updated.
|-
|[[apis/web-storage/properties/length|'''length''']]
|Retrieves the length of the key/value list.
|-
|[[apis/web-storage/properties/localStorage|'''localStorage''']]
|Retrieves  the Web Storage area specific to the current document.
|-
|[[apis/web-storage/properties/newValue|'''newValue''']]
|Gets  the new value of the [[apis/web-storage/methods/key|'''key''']].
|-
|[[apis/web-storage/properties/oldValue|'''oldValue''']]
|Gets  the previous value of the [[apis/web-storage/methods/key|'''key''']].
|-
|[[apis/web-storage/properties/remainingSpace|'''remainingSpace''']]
|Retrieves the remaining memory space, in bytes, for the storage object.
|-
|[[apis/web-storage/properties/sessionStorage|'''sessionStorage''']]
|Retrieves the Web Storage area for the session.
|-
|[[apis/web-storage/properties/storageArea|'''storageArea''']]
|Gets  the '''Storage''' object of the  affected document.
|-
|[[apis/web-storage/properties/url|'''url''']]
|Gets  the address of the document that  the update affects.
|}
