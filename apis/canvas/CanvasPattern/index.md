===Remarks===
To create a pattern from a portion of an image, create a temporary clip canvas and use [[canvas/methods/drawImage|'''drawImage''']]  to extract the piece.  Then, pass the tempoarary canvas to the [[canvas/methods/createPattern|'''createPattern''']] method.
