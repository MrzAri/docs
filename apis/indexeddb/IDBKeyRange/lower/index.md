===Remarks===
A key range is said to be ''open'' if the lower bound value is included in the range and ''closed'' if the range does not include the lower bound value.  Use the [[apis/indexedDB/properties/lowerOpen|'''lowerOpen''']] property to determine if the lower value is open or closed.
