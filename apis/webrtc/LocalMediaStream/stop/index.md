# stop

## Summary

Permanently halts the generation of data for the tracks' sources and removes the references to the sources.

*Method of [apis/webrtc/LocalMediaStream](/apis/webrtc/LocalMediaStream)*

## Syntax

``` {.js}
 localStream.stop();
```

## Return Value

No return value
