The
[[aria/attributes/aria-valuemin|'''aria-valuemin''']] and
'''aria-valuemax''' properties together specify the allowable range of an entry field, whereas the
[[aria/attributes/aria-valuenow|'''aria-valuenow''']] property specifies the value currently selected.
'''Note'''  For cross-browser compatibility, always use the WAI-ARIA attribute syntax to access and modify ARIA properties, for example <code>object.setAttribute("aria-valuenow", newValue)</code>.
